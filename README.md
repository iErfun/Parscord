## Parscord Config.txt
- config.txt
```ini
MODE=1  // 1 = Psiphon Mode , 2 = Warp Mode
DNS=1.1.1.1  // DNS address
REGION=US  // منطقه پروکسی ، فقط برای مود 1
```
